# CSC 405 Data Science Project - Synthetic Diabetes Prediction Dataset
## Group 5 - Adrian Aldridge, Eman Dweik, Sarah Robinson

## Dataset Information
The dataset used for this project is the Synthetic Diabetes Prediction Dataset, which is a designed dataset meant to simulate real-world diabetes-related health data. Synthetic datasets such as these are not collected from real participants but are simulated and used for machine learning, education, and research. In this project we will be performing exploratory analysis, hypothesis testing, and visualization of the dataset. <br>
Dataset Link: https://www.kaggle.com/datasets/miadul/synthetic-diabetes-prediction-dataset
